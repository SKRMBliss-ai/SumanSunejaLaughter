# Suman Suneja Laughter Hub: Reward System Documentation

## 1. Point Collection System
The application rewards users for engagement, consistency, and social sharing. Points are cumulative and contribute to both the user's "Level" and their ability to unlock shop items.

### Earning Actions & Values
| Action | Points Awarded | Description |
| :--- | :--- | :--- |
| **Daily Login** | **+20** | Awarded automatically upon opening the app for the first time each day. |
| **Maintain Streak** | **+20** | An additional bonus for logging in on consecutive days (keeping the streak alive). |
| **Share App** | **+20** | Awarded when the user initiates a "Share" action from the Rewards menu. |
| **Quick 1-Min Laugh** | **+20** | Completion of the guided 1-minute laughter exercise. |
| **Live Session** | **+30** | Participation in a live AI-guided laughter session. |
| **Rate My Laugh** | **+15** | Using the AI feedback tool to analyze laughter energy. |
| **Play Games** | **+10** | Playing any interactive game (e.g., Balloon Pop). |
| **Watch Video** | **+5** | Viewing a video resource from the library. |

---

## 2. Badge & Level System ("Joy Identity")
As users earn points, their public rank ("Joy Identity") evolves. This rank is displayed on their Profile and the global Leaderboard.

| Badge Title | Point Range | Status Meaning |
| :--- | :--- | :--- |
| **Smiling Starter** | **0 - 499** | **Beginner.** The starting rank for all new members. |
| **Giggle Generator** | **500 - 999** | **Active.** The user has started building a legitimate habit. |
| **Joyful Journeyer** | **1,000 - 1,999** | **Advanced.** The user is highly consistent and engaged. |
| **Laughter Master** | **2,000+** | **Elite.** The highest tier; meaningful dedication to the practice. |
